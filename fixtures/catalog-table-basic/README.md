# catalog-table-basic

This fixture validates the draft `cap.field.v1` and `cap.field_catalog.v1` shapes for the experimental `table-basic` Digest Pack.

It is intentionally small and mirrors the fields used by `fixtures/basic-table/`:

- `f1:table@shape#base`
- `f1:table@columns#compact`
- `f1:table@sample#k10`

This fixture does not test executable pack loading.
